Funkcje cyklonometryczne

Funkcje cyklonometryczne to funkcje odwrotne do funkcji trygonometrycznych. W funkcjach trygonometrycznych podajesz kąt L i obliczasz np. sin(L), cos(L) a w funkcjach cyklonometrycznych podajesz wartość funkcji trygonometrycznej i obliczasz kąt L, np. arcsin(x), arccos(x).

Można je określić tylko w tych przedziałach w których są różnowartościowe.

Wykres funkcji arcsin(x)

Jest to funkcja odwrotna do funkcji sinus, oznaczana jako arcsin(x). Y jest miarą łukową kąta z przedziału <-π/2, π/2>. Funkcja jest określona i ciągła dla x z przedziału <-1, 1>.

arcsin(x) = y
sin(y) = x
y ∈ <-π/2, π/2>
x ∈ <-1, 1>

Wykres funkcji arcsin(x) dla wartości w stopniach

var asinDegrees = x => Math.asin(x) * (180 / Math.PI)
arcsin(x) = y
y ∈ <-90, 90>
x ∈ <-1, 1>
Wykres funkcji arccos(x)

Jest to funkcja odwrotna do funkcji cosinus, oznaczana jako arccos(x). Y jest miarą łukową kąta z przedziału <0, π>. Funkcja jest określona i ciągła dla x z przedziału <-1, 1>.

arccos(x) = y
cos(y) = x
y ∈ <0, π>
x ∈ <-1, 1>
Wykres funkcji arccos(x) dla wartości w stopniach
var acosDegrees = x => Math.acos(x) * (180 / Math.PI)
arccos(x) = y
y ∈ <0, 180>
x ∈ <-1, 1>
Wykres funkcji arctan(x)

Jest to funkcja odwrotna do funkcji tangens, oznaczana jako arctan(x). Y jest miarą łukową kąta z przedziału <-π/2, π/2>. Funkcja jest określona i ciągła dla wszystkich x.

arctan(x) = y
tan(y) = x
y ∈ <-π/2, π/2>
x ∈ R
Wykres funkcji arctan(x) dla wartości w stopniach
var atanDegrees = x => Math.atan(x) * (180 / Math.PI)
arctan(x) = y
y ∈ <-90, 90>
x ∈ R
Wykres funkcji atan2(x, y)

Jest to funkcja odwrotna do funkcji tangens, oznaczana jako atan2(y, x) gdzie x i y to długości boków prostokątnego trójkąta. Funkcja zwraca kąt w radianach z przedziału <-π, π>.

Wykres gdy y = 1
